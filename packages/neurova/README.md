# neurova

The powerful TypeScript framework for AI-native apps. UI + Backend + AI, one install.

```bash
npm install neurova
```

```ts
import { ai, backend, core } from 'neurova'
// or, for tree-shaking:
import { chat } from 'neurova/ai'
import { Button, ChatWindow } from 'neurova/ui'
import { createServer } from 'neurova/backend'
import { id, logger } from 'neurova/core'
```
