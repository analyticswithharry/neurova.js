Post 2 — Why we built it
Most teams stitch together 6+ tools before writing product logic.

neurova.js flips that.

You get:
- @neurova/ui for frontend components
- @neurova/backend for Fastify-based APIs
- @neurova/ai for chat/embeddings/RAG/agents
- Shared primitives via @neurova/core

Everything designed to work together from day one.
